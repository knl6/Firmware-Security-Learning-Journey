# Day 4 Challenge 1: Temperature Converter

## What I Did
Wrote a program to convert Fahrenheit to Celsius

## Formula Used
C = (F - 32) * 5 / 9

## Code
challenge 1 solution



```
#include <stdio.h>
#include <stdlib.h>
#include <math.h>

int main() {
  int FTEMP = 98;



    printf("Farenheit:");
    printf ("%d\n", FTEMP);

    printf("Converted:");
    printf("%d\n", FTEMP = (FTEMP -32) * 5/9 );
    return 0;

}
```

Converted:36
